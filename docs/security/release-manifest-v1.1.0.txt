Package version: 1.1.0 security hardening
Package date: 2026-05-15
Manifest updated: 2026-05-15
License: Umbrella Public Access License
Rust toolchain: stable 1.95.0
Status: source-available package for non-commercial security and cryptographic
testing. Public FFI/client production bootstrap remains gated until production
platform attestation verifiers, mobile bridges, and server integration are
wired end to end.

Current hardening status: local Key Transparency split-view detection is now
implemented with public observations, verifiable equivocation evidence, strict
observation history, witness non-equivocation memory, and privacy-safe public
encoding. OPRF, backup, sealed sender, TLS/SPKI pinning, platform-verifier, and
test-only production-boundary checks have local fail-closed tests and audit
documents. Supply-chain hardening removes the unused optional hpke-rs libcrux
backend from root and fuzz lockfiles so RUSTSEC-2026-0124 is not hidden in the
SBOM. Real servers, real Android/iOS devices, live KT observation exchange, and
production witness channels remain release boundaries.

split-view теперь реализовано через публичные наблюдения, проверяемое
доказательство раздвоения, строгую историю наблюдений, память свидетеля и
безопасное для приватности публичное кодирование. OPRF, backup, sealed sender,

fail-closed тесты и аудит-документы. Настоящие серверы, настоящие Android/iOS
устройства, живой обмен KT-наблюдениями и боевые каналы свидетелей остаются
границами выпуска. Усиление цепочки зависимостей убирает неиспользуемый optional
libcrux-бэкенд hpke-rs из корневого и fuzz lockfile, чтобы RUSTSEC-2026-0124 не
